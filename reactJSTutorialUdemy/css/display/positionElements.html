Position HTML Elements

Static, Relative, Absolute, Fixed, Sticky

STATIC: default position value for HTML. RELATIVE: Position HTML elements relative to their normal position. It provides access to top, right, bottom and left properties. ABSOLUTE: It helps to position HTML element relative to the parent element. The parent element needs to have a position other than static or else the absolute element is positioned relative to the mail HTML element.
