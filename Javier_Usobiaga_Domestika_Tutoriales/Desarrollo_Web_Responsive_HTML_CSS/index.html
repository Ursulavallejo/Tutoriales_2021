Etiquetas de valor semantico:

Es una meta información que sirve para los navegadores y buscadores para entender nuestros contenidos y sabes cuáles son más importantes que otros.

em:

podemos dar énfasis a una palabra en concreto. Se renderiza en el navegador por defecto como una cursiva.

Strong:

O bien podemos dar importancia a una palabra en concreto. Se renderiza como una negrita.

Span:

SPAN nos permite agrupar una palabra o varias, trozos de texto, sin tener ningún valor semántico.

El span nos sirve para acotar palabras en concreto.
