Etiquetas de valor semántico:

Es una meta información que sirve para los navegadores y buscadores para entender nuestros contenidos y sabes cuáles son más importantes que otros.

Estas no producen un cambio de línea y están pensadas para trozos de texto.

em:

podemos dar énfasis a una palabra en concreto. Se renderiza en el navegador por defecto como una cursiva.

Strong:

O bien podemos dar importancia a una palabra en concreto. Se renderiza como una negrita.

Span:

SPAN nos permite agrupar una palabra o varias, trozos de texto, sin tener ningún valor semántico.

El span nos sirve para acotar palabras en concreto.

Etiquetas de Documento: son necesarios para que la página web muestre nuestro contenido correctamente.

Etiqueta HTML:

Se ponen al inicio y al final de nuestro documento HTML.

Etiqueta Body:

Nos permite indicar que parte de todo el documento HTML es el contenido como tal.

Etiqueta Head:

Nos permite incluir archivos que son meta información a archivos necesarios para mostrar la página, como puede ser la CSS o similar.

Etiqueta Title:

Le da un título a nuestra página y se ve en la pestaña del browser.

Etiqueta meta: Para que reconozca las tildes y caracteres no están en el alfabeto ingles se debe configurar en el head para que pueda leer la codificación.

meta charset="utf-8" ( no es necesario cerrar esta etiqueta).

Etiqueta !DOCTYPE html: Va al inicio del proyecto y no se cierra esta etiqueta.

Permite al navegador saber que tiene que renderizar la web siguiendo los estándares actuales.

Ejemplo de lo que debe incluir una página web:

    
        < !DOCTYPE html >
        < html lang="en" >
        < head >
           < meta charset="UTF-8" >
        < title>Titulo WEB< /title >
        < /head>
        < body>
        < h1>Contenido< /h1>
        < /body>
        < /html>

Comentarios es HTML: < !-- TEXTO comentario -- > ( sin espacios )
