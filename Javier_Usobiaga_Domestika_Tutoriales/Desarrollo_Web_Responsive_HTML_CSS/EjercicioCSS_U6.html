El diseño responsive y Mobile First

El diseño responsive se basa en una estructura flexible, basada en porcentajes, y en unas reglas especiales de CSS que se llaman media queries.

Para crear uan estructura flexible nos podemos basar con porcentajes, los widths, las anchuras que se asignen a las columnas deben estar en porcentajes en lugar de píxeles.

Las media queries:

Las media queries son una serie de reglas condicionales. Podemos hacer que algunos estilos sólo se ejecuten si se cumple una condición, como puede ser un tamaño mínimo de pantalla. Esto nos permite hacer que dos cajas floten una al lado de otra cuando tengan espacio suficiente para mostrarse.

@media screen and (max-width: 800px){ estilos }

Navegación responsive

El problema de la navegación en la pantalla pequeña es que un montón de enlaces puestos uno debajo de otro pueden entorpecer el acceso al contenido. La mayor parte de navegaciones responsive cuentan con ocultar de alguna manera la navegación en móviles y mostrarla sólo cuando el usuario la necesita.
