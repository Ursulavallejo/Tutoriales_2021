Estilos tipográficos

En un diseño web, el tratamiento tipográfico es fundamental para encontrar un tono de voz y transmitir adecuadamente un mensaje. Tamaño, peso, altura de línea o colo

Font-size:

El font size es el único atributo que se hereda, si el div tiene por ejemplo 20 px el contenido de este lo heredara

Medidas:

Pixel es la mas habitual.

Em es una unidad relativa al font-size heredado. Por ejemplo, si decimos que nuestro Párrafo tenga 1.5em, es decir, 1.5 veces el tamaño heredado. Si el div tiene 20 px. El párrafo tendrá 1.5 veces los 20 pixeles, es decir, treinta pixeles.

Font-weight:

Es el peso tipográfico, por ejemplo el font-weight: bold, carga la version negrilla. Si se quiere resetear se puede usa font-weight: normal.

También se puede especificar el peso con números: font-weight: 200, 400, 600... se denomina peso equivalente.

font-style: italic carga la cursiva

font-style: normal resetea los estilos

Para subrayar una tipografía se usa el text-decoration : underline

Para tachar una tipografía se usa el text-decoration : line-through

Para resetear text-decoration : none

Altura de linea : line-height

line-height : 1.5,

Se puede establecer en unidades absolutas como píxeles o, en este caso, se nos permite una unidad relativa al tamaño tipográfico, por lo tanto no hace falta especificar nada sino que podemos poner 1.5. Que significa 1.5 veces el tamaño asignado tipográfico.

Aplicar color:

Palabra clave

Colores asignados a palabras clave en inglés, como red, orange, black, white o pink.

Ejemplo: Este texto tiene aplicado color: hotpink;

Hexadecimal

Los colores hexadecimales se preceden con #, como por ejemplo: #000000 (negro), #FFFFFF (blanco), #FF0000 (rojo).

Ejemplo: Este texto tiene aplicado color: #FF0000;

rgb

La declaración rgb se pone entre paréntesis (sin espacio entre los paréntesis y la palabra rgb), y se declara de 0 a 255 los valores de rojo, verde y azul. Ejemplo: rgba(255, 0, 0)

Ejemplo: Este texto tiene aplicado color: rgba(255, 0, 0);

rgba

La declaración de rgba es similar a rgb, pero se añade un cuarto valor, en base a 1, que indica la opacidad. Ejemplo: rgba(0, 0, 0, 0.4)

Ejemplo: Este texto tiene aplicado color: rgba(255, 0, 0, 0.4);
