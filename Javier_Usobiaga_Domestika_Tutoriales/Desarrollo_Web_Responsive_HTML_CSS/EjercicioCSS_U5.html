* Fondos e imágenes de fondo

background-color

Fondo de color:

Se utiliza background-color para poner un color base al elemento.

Background color

background-image

Para agregar una imagen se utiliza background-image. Por ejemplo si se pone en una superficie muy grande esta se repetirá, para que no pase esto se utiliza:< background-repeat: no-repeat >.

Imagen fondo

Background-position

* Background-position: right top (o también por pixeles); corresponde izquierda arriba)// para posicionar la imagen dentro de alguna parte del body

background-position: 20px 30px //corresponde izquierda arriba


background-position: right 20px bottom 30px; Para posicionar desde la derecha


Background

La propiedad Background agrupa todas las anteriores / background: hotpink url(img/..) no-repeat right top //

La imagen aparecerá con su tamaño, pero se puede modificar con: background-size: X Y


Ejemplo de background dentro de un div:


background-size: contain

A veces no queremos que las imágenes se redimensionen en un tamaño concreto, sino que tomen el tamaño de su contenedor. Para esto utilizamos la propiedad: background-size: contain.

Quiere decir que la imagen se contenga completamente dentro de su contenedor.

Ejemplo de imagen background dentro de su contenedor: /background-size: contain/


background-size: cover

Se utiliza cuando queremos que la imagen cubra completamente el contenedor

Ejemplo de imagen background dentro de su contenedor: /background-size: cover/

font-family

Si trabajamos con tipografías del sistema podemos asignar un stack (orden de preferencia de tipografías)

Si tenemos un nombre de tipografía que tiene espacios se pone con comillas / font-family: "Time New Roman" , serif / tipografía con remates
